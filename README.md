# Cyclistic Bike Share Analysis

An urban mobility analysis of Cyclistic bike-share usage conducted as part of the **Google Data Analytics Professional Certification (Coursera)**. This project explores behavioral differences between casual riders and annual members using data-driven methods in **R** and **Tableau**.

---

## Project Overview

**Objective:**  
Identify how casual riders and annual members use Cyclistic bikes differently, and propose data-driven strategies to increase annual membership.

**Methods & Tools:**
- Data cleaning and transformation using **R (tidyverse)**
- Exploratory data analysis and visualization
- Aggregation of 800,000+ trip records
- Interactive dashboards built in **Tableau**
- Business-oriented insights and recommendations

## Key Insights (Summary)

- Casual riders use bikes primarily on weekends and afternoons, indicating leisure-based usage.
- Annual members ride more frequently on weekdays, suggesting commuting behavior.
- Membership conversion strategies should target high-traffic leisure zones with seasonal campaigns.
